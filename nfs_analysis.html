🔑 ハッシュアルゴリズム

FNV-1 64bit 変種 ✅ Ghidra 解析で確定

# FUN_140d023a0: 1バイトハッシュ
def hash_char(byte, seed):
    return (seed * 0x1000193 ^ byte) & 0xFFFFFFFFFFFFFFFF

# FUN_140d023c0: ファイルハッシュ計算
def calc_hash(filename, path):
    h = 0
    for b in filename.lower():  h = hash_char(b, h)
    for b in path.lower():      h = hash_char(b, h)
    return h

パス正規化ルール (FUN_140d02b60)

  • \ → / に変換
  • // → / に正規化
  • 先頭の ./ を除去
  • 末尾の / を除去
  • 全て小文字に変換
  • ハッシュは filename + path の順で計算（path が後）
